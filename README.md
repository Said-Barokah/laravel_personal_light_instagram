# Proyek Laravel 11 Anda

Deskripsi singkat tentang proyek ini dan tujuannya.

## Persyaratan Sistem

- **PHP** >= 8.2
- **Composer** >= 2.0
- **Node.js** >= 16.0
- **Database** (MySQL, PostgreSQL, atau sesuai kebutuhan)
